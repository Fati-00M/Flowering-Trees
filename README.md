# 🌳 Flowering Trees Dataset & Climate Traits Analysis

## 📌 Overview
This project builds a structured dataset of flowering tree species and their ecological traits using data from botanical databases and open sources. It focuses on linking plant biodiversity with climate-related variables such as native range, flowering patterns, and environmental conditions.

The goal is to support climate and biodiversity analysis through a clean, usable dataset that can be further expanded for GIS and machine learning applications.

---

## 📊 Project Components

### 1. Dataset Construction
- Species list compiled from taxonomic backbones (e.g., World Flora Online)
- Trait extraction from sources such as Wikipedia and plant databases
- Focus on flowering-related characteristics:
  - Flowering season
  - Native range
  - Growth conditions
  - Taxonomic classification

### 2. Data Processing
- Cleaning and standardization of species names
- Handling missing or inconsistent trait data
- Creation of a structured trait matrix (`tree_trait_matrix.csv`)

### 3. Analysis Notebook
- Exploratory data analysis in Jupyter Notebook (`flowering_traits.ipynb`)
- Assessment of data completeness
- Trait distribution analysis
